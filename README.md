## Descripción del Trabajo
Este trabajo práctico consiste en el desarrollo de una base de datos para un sistema de centro médico, incluyendo tanto el diseño del modelo como la implementación de consultas SQL.
El trabajo se divide en dos partes principales:

## Parte 1 - Modelo de Datos
En esta parte se diseñó el modelo entidad-relación del sistema, representando las entidades principales como pacientes, médicos, consultas y recetas, junto con sus relaciones.


## Parte 2 - Consultas SQL
En esta sección se desarrollaron distintas consultas SQL para operar sobre la base de datos, incluyendo:

- Creación de índices
- Uso de vistas
- Consultas con filtros
- Actualizaciones de datos
- Consultas con JOIN
- Subconsultas
- Agrupamientos y conteos
